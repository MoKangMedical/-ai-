# 笔目鱼网站设计关键要素

## 视觉设计特征

### 配色方案
- **主色调**：紫色系（#6B46C1 到 #D946EF 渐变）
- **背景**：深紫色渐变，带有网格纹理
- **强调色**：粉紫色、亮紫色
- **文字**：白色主文字，紫色标题强调
- **按钮**：紫色渐变，带圆角和阴影

### 布局特点
1. **固定顶部导航**：黑色半透明背景，白色文字
2. **Hero区域**：全屏高度，中心对齐，大标题+功能标签+CTA按钮+产品预览
3. **功能展示**：左右交替布局，序号+标题+描述+截图
4. **编辑器界面展示**：带圆角边框的产品截图，展示实际使用场景

### 交互元素
- 圆角按钮（立即体验、免费使用）
- 悬停效果
- 平滑滚动
- 功能标签展示（横向排列的小标签）

### 右侧悬浮工具栏
从截图中可以看到右侧有多个悬浮按钮：
- 客服
- 反馈
- 文档
- 其他功能入口

## 核心功能界面

### 文本编辑器
- 左侧：编辑区域
- 右侧：功能面板（翻译引擎选择、语法校对、时态校对等）
- 顶部：工具栏（文档操作、格式设置）
- 支持全文/选中文本处理
- 实时AI处理反馈

### AI功能展示
- 学科润色：显示修改前后对比
- 通用润色：显示优化建议
- DeepSeek-R1 通用版标识

## 技术栈建议

基于观察到的功能，推荐使用：
- **前端框架**：React + TypeScript
- **UI组件**：TailwindCSS（用于快速实现渐变和响应式）
- **文本编辑器**：富文本编辑器组件
- **AI集成**：Kimi 2.5 API
- **数据库**：需要用户系统和文档存储
- **后端**：需要API服务处理AI请求

## 需要实现的核心页面

1. **首页**（Landing Page）
   - Hero区域
   - 功能展示区
   - 合作伙伴/认证展示
   - Footer

2. **编辑器页面**（主应用）
   - 文本编辑器
   - AI功能面板
   - 工具栏
   - 实时处理

3. **用户系统**
   - 登录/注册
   - 会员管理
